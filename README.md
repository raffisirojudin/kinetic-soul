## 📌 Tentang Kinetic Soul
Kinetic Soul adalah proyek eksperimen visual interaktif yang mensimulasikan gerakan elemen-elemen alam (seperti partikel air, api, atau angin) di dalam browser. Proyek ini menggabungkan matematika fisika sederhana dengan manipulasi grafis web.

### ✨ Fitur Utama
* **Element Manipulation:** Pengguna bisa memilih jenis elemen dan melihat bagaimana partikel bereaksi di layar.
* **Cursor Interaction:** Partikel akan mengikuti, menjauh, atau meledak berdasarkan pergerakan dan klik kursor *mouse* pengguna.
* **Physics Simulation:** Menerapkan efek gravitasi, gesekan, dan pantulan (*bounce*) pada setiap partikel.
* **Performance Control:** Pengaturan jumlah partikel untuk menyesuaikan dengan performa perangkat pengguna.

### 🛠️ Tech Stack
* **Core:** HTML5 Canvas API, JavaScript (RequestAnimationFrame untuk animasi halus), CSS3.
